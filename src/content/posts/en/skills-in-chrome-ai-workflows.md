Google Chrome is launching Skills, which lets users save and reuse their most helpful AI prompts and run them with a single click. This solves the pain point of re-entering the same prompts repeatedly.

### Build One-Click Workflows

When you write a prompt you'll want to use again, you can save it as a Skill directly from chat history. Next time, select your saved Skill in Gemini in Chrome by typing forward slash (/) or clicking the plus (+) button, and your Skill will run on the current page along with any other selected tabs. Edit or create new Skills anytime.

### Use Case Examples

Early testers have used Skills for various tasks:

- **Health & Wellness**: Quickly calculating protein macros for any recipe
- **Shopping**: Generating side-by-side spec comparisons across multiple tabs
- **Productivity**: Scanning lengthy documents for important information

### Skills Library

Google is also launching a library of ready-to-use Skills for common tasks. For example, analyzing product ingredients or selecting the perfect gift by cross-referencing your budget with the recipient's interests. Save and try any Skill, or edit the prompt to customize.

### Security & Privacy

Skills are built on Chrome's foundation of security and privacy, using the same safeguards as Gemini. They ask for confirmation before certain actions like adding calendar events or sending emails, and benefit from Chrome's layered protections including automated red-teaming and auto-update capabilities.
