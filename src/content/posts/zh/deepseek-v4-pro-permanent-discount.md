DeepSeek 已将 V4 Pro API 的 75% 折扣定价调整为永久价格，此举显著改变了 AI 推理 API 市场的竞争格局。

调整后，DeepSeek V4 Pro（拥有 1M 上下文窗口和 384K 最大输出）的价格为：
- 输入缓存命中：$0.003625/M token
- 输入缓存未命中：$0.435/M token
- 输出 token：$0.87/M token

这相当于原始定价的 1/4，最终将在 5 月 31 日促销期结束后正式生效。V4 Flash 也保持了极具竞争力的价格：输出仅 $0.28/M token。

DeepSeek 的定价逻辑揭示了一个重要趋势：模型厂商正在用「永久低价」锁定开发者生态。当推理成本持续下降，开发者选择模型的主要考量从「谁更便宜」转向「谁在工具链中更好用」。DeepSeek 已经适配了 Claude Code、OpenAgents 等多个主流 Agent 框架，配合这个定价策略，其平台吸引力正在从「性价比选项」升级为「默认选项」。

在一个以推理成本为核心的竞争时代，DeepSeek 这次定价调整可能比新款模型发布更具战略意义。它不仅降低了应用层开发者的运营门槛，也在迫使其他模型提供商重新思考自己的定价结构。
