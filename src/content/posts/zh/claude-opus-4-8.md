Anthropic 今日发布 Claude Opus 4.8，这是其旗舰级 Opus 系列的最新版本，在多项基准测试中实现了显著提升，并同步推出了两项重要的产品功能：动态工作流（Dynamic Workflows）和努力控制（Effort Control）。

## 性能全面提升

Opus 4.8 在编码、智能体任务和专业知识工作方面均超越前代 Opus 4.7。多位早期测试者报告了显著的改进：Cursor 测试显示工具调用效率更高，使用更少的步骤完成同样的任务；Databricks 的 Genie AI 智能体报告称，Opus 4.8 在智能推理和多步骤问题处理上实现了质的飞跃。

最值得注意的是模型在"诚实度"上的提升。Anthropic 的评估显示，Opus 4.8 对自身工作中的缺陷保持沉默的可能性比前代降低了约四倍。它更倾向于主动标记不确定性，而非做出无法支撑的自信声明。

## 智能体能力跃升

多家测试者在智能体任务中报告了实质性改进。法律 AI 平台报道 Opus 4.8 在其法律智能体基准测试中取得了历史最高分。浏览器智能体测试 Online-Mind2Web 上，Opus 4.8 达到了 84% 的得分率，超过了 GPT-5.5。Devin 报告称工具使用更加清晰、指令遵循更加一致。

## 两项重要新功能

**动态工作流**是 Claude Code 的研究预览功能，允许 Claude 在单个会话中规划工作并运行数百个并行子智能体，甚至能自主完成跨越数十万行代码的代码库迁移任务——从启动到合并，以现有测试套件为质量门槛。

**努力控制**让用户可以在 claude.ai 上选择 Claude 投入多少"努力"到回应中。高努力模式下模型会进行更深入的思考以获得更优结果，低努力模式下则回应更快、消耗更少配额。

此外，Messages API 现在支持在消息数组中传递系统提示，开发者可以在智能体运行过程中动态更新权限、Token 预算或环境上下文，而无需破坏提示缓存。

## 定价与可用性

Opus 4.8 今日起全面可用，价格与 Opus 4.7 保持一致：每百万输入 Token $5，每百万输出 Token $25。快速模式（2.5 倍速度）价格降至每百万 Token $10/$50。开发者可通过 API 使用 `claude-opus-4-8` 模型标识符。

Anthropic 同时预告了"Project Glasswing"项目下的 Mythos 级模型，这些模型拥有比 Opus 更高的智能水平，目前正由部分组织用于网络安全工作，预计将在未来几周内向所有用户开放。
