Anthropic 近日宣布收购 Stainless，一家专注于 API SDK 生成和 MCP（Model Context Protocol）服务器工具的公司。

Stainless 自 2022 年成立以来，一直为 Anthropic 所有官方 SDK 提供底层技术支持，覆盖 TypeScript、Python、Go、Java、Kotlin 等主流语言。超过数百家公司依赖 Stainless 生成 SDK、CLI 工具和 MCP 服务器——这些是让开发者和智能体调用 API 的关键连接件。

Anthropic 平台工程主管 Katelyn Lesse 表示："智能体的价值取决于它们能连接到什么。把 Stainless 团队引入 Anthropic，将推进 Claude 连接数据和工具的能力。"

这笔收购的核心信号十分明确：Anthropic 正在从"模型公司"向"智能体平台公司"转变。MCP 协议是 Claude 接入外部系统的桥梁，而 Stainless 恰好是这座桥梁的核心建造者。将 SDK 和 MCP 工具链内化，意味着 Anthropic 可以更快速地为 Claude 推出新的 API 集成，降低第三方开发者接入的门槛。

在 Google 推广 A2A（Agent-to-Agent）协议、OpenAI 也在加码工具使用的背景下，Anthropic 对 MCP 生态的深度整合显得尤为关键。谁控制了智能体的"连接层"，谁就掌握了智能体经济的底层基础设施。
