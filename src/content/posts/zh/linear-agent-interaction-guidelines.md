Linear 发布了 Agent Interaction Guidelines (AIG)，一套关于 AI 智能体与人机交互的基础设计原则。随着智能体改变软件的规划、构建、审查和部署方式，新的交互契约变得必要。

## 核心原则

### 1. 智能体应始终披露其身份

当人类与智能体并肩工作时，人类需要立即确定自己在与谁交互。智能体必须清晰标识自己的身份，确保不会被误认为真人。

### 2. 智能体应原生融入平台

智能体默认应能通过现有 UI 模式和平台的标准操作来工作，就像人类用户一样使用相同的功能。

### 3. 智能体应提供即时反馈

沉默会导致不确定性。被调用时，智能体应立即提供不打扰的反馈，让用户知道请求已被接收。

### 4. 智能体应清晰透明地展示内部状态

智能体应清楚表明自己是正在思考、等待输入、执行中还是已完成。人类应能一目了然地理解发生了什么，并在需要时检查底层推理、工具调用和决策逻辑。

### 5. 智能体应尊重退出请求

当被要求退出时，智能体应立即后退，只有在收到明确信号后才重新参与。

### 6. 智能体不能被问责

人类与智能体之间应有清晰的委托模型。智能体可以执行任务，但最终责任应始终由人类承担。
