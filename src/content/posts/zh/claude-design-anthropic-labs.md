今天，Anthropic Labs 正式推出 Claude Design，这是一款基于 Claude Opus 4.7 视觉模型的协作式设计工具。用户可以通过自然语言描述需求，Claude 会立即生成第一版设计，然后通过对话、评论、直接编辑或自定义滑块进行迭代优化。

Claude Design 支持多种场景：

- **真实原型**：设计师可将静态 mockup 转为可交互原型，无需代码审查
- **产品线框图**：产品经理快速绘制功能流程，直接交给 Claude Code 实现
- **设计探索**：快速生成多个设计方向进行对比
- **演示文稿**：几分钟内从大纲生成完整 PPTX 或 Canva 文件
- **营销物料**：创建落地页、社交媒体素材和 campaign 视觉

当接入设计系统后，Claude 能自动应用品牌规范，确保输出与公司其他设计保持一致。目前已在 Claude Pro、Max、Team 和 Enterprise 计划中逐步开放。
