前 Django 联合创始人、长期关注 AI 的评论者 Simon Willison 发表了一篇数据密集的分析文章，指出 2026 年 4 月是一个新的转折点：Anthropic 和 OpenAI 终于找到了真正的产品市场契合点，而驱动这一切的是编码智能体。

证据非常具体。两家公司都将企业定价从固定席位费转向了基于 API token 消耗的模式。Anthropic 的企业方案变为每席位 20 美元外加 API 使用费，OpenAI 在 4 月也做了类似的调整。重度用户使用 Claude Code 每月消耗约 1200 美元的 token——比普通消费者订阅高出数个数量级。

Willison 认为，那些广为流传的"AI 预算超支"故事实际上支持而非否定产品市场契合的论点。Uber 的 CTO 向投资者透露，公司全年 AI 预算在短短几个月内就已用尽，主要来自 Claude Code。微软则通过取消 Claude Code 许可来控制支出。经典的定价经验法则在此适用：如果客户一边倒吸一口凉气一边说"好"，说明价格点找对了。

最引人注目的数据来自 SpaceX 意外提交的 S-1 文件，其中披露 Anthropic 已承诺每月支付 12.5 亿美元用于推理算力，合同持续至 2029 年 5 月。仅这一家供应商的合同就揭示了前沿实验室当前的推理需求规模。

据传 Anthropic 有望实现 109 亿美元的季度营收，并首次实现盈利。两家公司正在大力招聘企业销售岗位——OpenAI 的 32% 和 Anthropic 的 27% 的空缺职位都属于企业销售方向。

更深层的意义在于战略层面。通过将企业定价转向 API token 消耗并绕过 Cursor 和 GitHub Copilot 等中间商，前沿实验室正在捕获自身基础设施的全部价值。Willison 将此刻称为 2026 年 4 月转折点，紧随 2025 年 11 月智能体首次真正可用的转折之后。即将到来的 IPO 的 S-1 文件将告诉我们营收数据是否与叙事相符。
