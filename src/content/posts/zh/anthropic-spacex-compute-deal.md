Anthropic 今日宣布与 SpaceX 达成一项重要的计算基础设施合作协议。根据协议，Anthropic 将获得 SpaceX 旗下 Colossus 1 数据中心的所有算力——超过 300 兆瓦的新增容量，包含逾 220,000 张 NVIDIA GPU，预计一个月内上线。

这笔交易不仅是为了扩容。它标志着 AI 公司对计算资源的需求已经从传统云厂商扩展到更广泛的工业与能源基础设施领域。

伴随这一合作，Anthropic 同步提升了多项产品使用限制：

- **Claude Code**：Pro、Max、Team 及 Enterprise 计划的五小时间率限制翻倍；
- **高峰时段限制取消**：Pro 和 Max 账户不再受高峰时段 Claude Code 使用限制；
- **API 提升**：Claude Opus 模型的 API 频率限制大幅提高。

这是 Anthropic 一系列大型计算投资的最新一环。过去一年内，该公司已宣布：

- 与 Amazon 达成最高 5 GW 的计算合作，其中近 1 GW 将在 2026 年底前上线；
- 与 Google 和 Broadcom 达成 5 GW 协议，预计 2027 年开始交付；
- 与 Microsoft 和 NVIDIA 建立战略合作伙伴关系，包括 300 亿美元的 Azure 算力；
- 通过 Fluidstack 投资 500 亿美元建设美国 AI 基础设施。

更值得注意的是，Anthropic 在声明中表示，已表达与 SpaceX 合作开发多吉瓦级轨道 AI 计算能力的兴趣——这是 AI 计算走向太空的潜在信号。

从 Agent Economy 的视角看，这一系列布局表明：AI 公司正从"租用云 GPU"的模式转向"自己控制基础设施"的模式。计算已经从运营支出变成战略资产，而谁掌握算力，谁就掌握了下一次模型能力跃迁的钥匙。
