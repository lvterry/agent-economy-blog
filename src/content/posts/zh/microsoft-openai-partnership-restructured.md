OpenAI 与微软的合作关系今天迎来重大转折。双方联合宣布修订合作协议，核心变化是微软放弃了对 OpenAI 模型的独家云销售权，作为交换，微软不再向 OpenAI 支付收入分成。

新协议的关键条款：

- **非独家授权**：微软将继续持有 OpenAI 知识产权的许可，有效期至 2032 年，但从此转为非独家。这意味着 OpenAI 可以将其模型和产品部署在 AWS、Google Cloud 等任何云平台上。
- **Azure 优先**：微软仍然是 OpenAI 的主要云合作伙伴。OpenAI 产品将优先在 Azure 上发布，除非微软无法或不支持所需的基础设施能力。
- **收入分成终结**：微软不再向 OpenAI 支付收入分成。反过来，OpenAI 向微软的收入分成将继续至 2030 年，比例不变但有总额上限。
- **微软仍是主要股东**：微软将继续直接参与 OpenAI 的成长。

这个时间点很耐人寻味。就在上周，OpenAI 刚刚发布了 GPT-5.5，企业市场对 AI 模型的需求正在从实验阶段转向规模化部署。OpenAI 获得多云自由后，可以同时向更多企业客户销售，不再被锁定在微软的生态里。

对微软来说，放弃独家权意味着 Azure AI 的差异化优势被削弱——Azure 客户不能再声称"只有这里能买到 OpenAI"。好处是微软省下了收入分成成本，而且 OpenAI 的增长仍然通过股东身份给微软带来回报。

对整个行业而言，这意味着 AI 模型层的竞争格局进一步分散化。Anthropic 的 Claude、Google 的 Gemini、Meta 的 Llama 以及 OpenAI 的 GPT 系列，将在同一个多云竞技场上争夺企业客户。OpenAI 的"多云策略"可能会加速企业客户对 AI 模型的采购灵活性需求。
