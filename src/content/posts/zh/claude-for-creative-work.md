Anthropic 今天发布了一系列针对创意专业人士的 Claude 连接器（connectors），合作伙伴包括 Blender、Autodesk、Adobe、Ableton 和 Splice 等八家创意工具厂商。这些连接器基于 MCP（Model Context Protocol）协议构建，让 Claude 可以直接在这些专业软件中执行操作、查询文档和自动化工作流。

**八款连接器覆盖了创意行业的各个领域：**

Blender 的 MCP 连接器允许用户通过自然语言与 Blender 的 Python API 交互，分析场景结构、批量修改对象属性，甚至直接在 Blender 界面中添加自定义工具。Anthropic 同时宣布加入 Blender Development Fund 作为赞助方，以支持 Blender Python API 的持续开发。由于基于开放的 MCP 标准，该连接器也兼容其他大语言模型。

Adobe 的连接器覆盖了 Photoshop、Premiere、Express 等 50 多种 Creative Cloud 应用，用户可以通过 Claude 处理图片、视频和设计素材。Autodesk Fusion 的连接器允许工程师通过对话创建和修改 3D 模型。Ableton 连接器让 Claude 能基于官方产品文档回答 Live 和 Push 的使用问题。Splice 的连接器则帮助音乐制作人在 Claude 内搜索免版税样本库。

此外，Anthropic 还与罗德岛设计学院（RISD）、瑞林艺术与设计学院和伦敦大学金史密斯学院合作，为艺术与计算相关课程提供 Claude 和连接器访问权限。

**信号解读：**

这次发布有几个值得关注的信号。首先，MCP 作为连接标准正在被主流专业软件厂商接受——Blender 甚至主动为其开发了官方连接器，这在开源社区是一个重要进展。其次，Anthropic 选择了创意工具作为 MCP 生态的重点突破口，而非传统的企业软件场景，说明他们看到了 AI 在创意生产中的差异化价值。第三，与设计院校的合作表明 Anthropic 在系统性地培养下一代创意工作者的 AI 使用习惯。

对于关注 AI 应用落地的人来说，这篇文章展示了 MCP 协议从基础设施走向真实产品的关键一步——当 AI 助手能直接操控 Blender 场景或 Ableton 音轨时，创意流程的自动化才真正开始上量。
